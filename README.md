# Expense Tracker App

A Flutter-based mobile application developed as a student project to help users track their daily expenses.  
The app lets users add expenses, assign categories, manage transaction records, and view a chart summary of spending.

## Features

- Add and save expenses
- Choose expense categories
- Display a list of recorded transactions
- Remove expenses from the list
- Show a chart summary for spending visualization
- Clean and simple user interface

## Tools and Technologies

- Flutter
- Dart
- Material Design

## What I Worked On

- Developed the user interface using Flutter widgets
- Built the form for entering new expenses
- Added dropdown category selection
- Implemented expense list management
- Displayed expense data visually with a chart
- Used `setState()` for updating app data and screen content

## Purpose

This project was created to improve my skills in Flutter, mobile UI design, and handling dynamic user data in an application.
